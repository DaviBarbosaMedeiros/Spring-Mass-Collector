# Spring-Mass Analyzer

Open-source device for spring-mass experiment analysis.

## Introduction

The main objective of this project is to develop a device capable of analyzing data obtained from a spring–mass experiment.

To enhance accessibility, it is necessary to design a low-cost solution using 3D printing and a microcontroller.

Finally, after completing the physical device, the project will be made open source, accompanied by comprehensive documentation to ensure its proper use and reproducibility.

## Project Overview

This project aims to develop a low-cost device capable of analyzing data obtained from a spring-mass experiment using:

- ESP32
- Distance Sensor
- LCD Display
- 3D Printed Structure

The complete system is designed to be fully open source.
